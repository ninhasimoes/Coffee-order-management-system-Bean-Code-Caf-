# ☕ Bean & Code Café — Order Management System

This project is a Java-based console application that simulates an order management system for a café. It was developed as a practice exercise for Object-Oriented Programming (OOP) concepts.

---

## 🎯 Objective

The main goal of this project is to practice and reinforce the following OOP concepts:

- 🧩 **Class Composition** — a class contains references to objects of another class  
- 🏗️ **Constructors** — proper initialization of objects with and without parameters  
- 🔒 **Encapsulation** — private attributes accessed through getters and setters  
- 📦 **Arrays of Objects** — storage and manipulation of object collections  

---

## ⚙️ Features

The system allows users to:

- Create new orders  
- Add items to an order  
- Finalize orders  
- List open orders  
- Calculate daily revenue  

---

## 🏗️ Project Structure

The system is composed of the following classes:

- `Main` → controls the application flow (menu system)  
- `Order` → represents a customer order  
- `OrderItem` → represents items inside an order  
- `Customer` → stores customer information  

---

## 🧠 Key Concepts Applied

- Object-Oriented Programming (OOP)  
- Arrays of objects  
- Method reuse and modularization  
- Menu-driven console application  
- Separation of responsibilities between classes
